# Калькулятор зарплаты

Интерактивный веб-сайт для расчёта "ожидаемой" зарплаты на основе отработанных дней и дополнительных начислений.

## Возможности

- Интерактивный календарь для отметки рабочих дней
- Три типа дней: полный (1), неполный (0.5), выходной (0)
- Добавление доплат к конкретным рабочим дням
- Автоматический расчёт общей суммы за месяц
- Статистика по месяцам
- Настройка базовой ставки и валюты
- Экспорт и импорт данных в JSON
- Сохранение данных в браузере (localStorage)

## Установка

1. Убедитесь, что у вас установлен Node.js (версия 14 или выше)
2. Клонируйте репозиторий или скачайте файлы проекта
3. Установите зависимости:
   ```bash
   npm install
   ```

## Запуск

Для запуска в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

Для сборки production-версии:
```bash
npm run build
```

## Использование

1. **Отметка дней**: Нажимайте на ячейку календаря, чтобы переключать тип дня:
   - Серый → Зелёный (полный день)
   - Зелёный → Жёлтый (неполный день)
   - Жёлтый → Серый (выходной)

2. **Добавление доплат**: Нажимайте кнопку "+" на ячейке дня, чтобы добавить доплату

3. **Настройки**: Изменяйте базовую ставку и валюту в разделе "Настройки"

4. **Навигация**: Используйте стрелки для переключения месяцев или кнопку "Сегодня"

5. **Управление данными**:
   - Экспорт: Сохраняет данные в JSON-файл
   - Импорт: Загружает данные из JSON-файла
   - Очистка: Удаляет все данные (с подтверждением)
